# Superteam Academy LMS - Demo Video Script

*Target length: 2-3 minutes*

## Opening (10 seconds)

"Introducing Superteam Academy - a decentralized learning platform built for the Solana ecosystem, designed specifically for Superteam Brazil."

## Landing Page Tour (20 seconds)

- Show the hero section with the CTA
- Demonstrate language switcher (Portuguese, English, Spanish)
- Scroll to show featured courses and gamification elements
- "The platform is fully localized for the Brazilian market while supporting the global community."

## Course Catalog (20 seconds)

- Click "View Courses" to enter catalog
- Show course cards with difficulty levels (Beginner, Intermediate, Advanced)
- Filter/search functionality
- "Courses are organized by skill level with clear progression paths."

## Wallet Connection (15 seconds)

- Click "Connect Wallet" button
- Show wallet adapter popup with supported wallets
- "Seamlessly connect your Solana wallet to track progress and earn on-chain credentials."

## Course Detail Page (20 seconds)

- Click into a course (e.g., "Building on Solana")
- Show course overview, modules, and lesson list
- Highlight XP rewards visible for each lesson
- "Each course is broken into bite-sized modules with clear XP rewards."

## Interactive Lesson (30 seconds)

- Open a coding lesson
- Show the Monaco code editor with syntax highlighting
- Demonstrate editing code in the editor
- "The built-in VS Code-like editor lets you write and test Solana code directly in your browser."
- Show the run/submit functionality

## Gamification Features (20 seconds)

- Navigate to the user dashboard
- Show XP counter, streak display, level progress
- Click to the leaderboard
- "Gamification keeps learners engaged with XP points, daily streaks, and competitive leaderboards."

## Profile & Settings (15 seconds)

- Show profile page with achievements
- Settings page with preferences
- "Users can customize their experience and track their learning journey."

## Closing (15 seconds)

- Return to landing page
- Show responsive design (mention mobile support)
- "Superteam Academy - empowering the next generation of Solana developers in Brazil and beyond. Built with Next.js, Solana Wallet Adapter, and shadcn/ui for a modern, accessible learning experience."

## Key Points to Emphasize

1. **Full-stack implementation** - Not just mockups, a working dApp
2. **Multi-language support** - Portuguese, English, Spanish
3. **Solana integration** - Real wallet connection, on-chain credentials
4. **Modern tech stack** - Next.js 16, TypeScript, Tailwind
5. **Content management** - Sanity CMS integration for course updates
6. **Gamification** - XP, streaks, leaderboards for engagement
7. **Mobile-responsive** - Works on all devices

## Recording Tips

- Use 1920x1080 resolution
- Zoom browser to 110-125% for visibility
- Smooth, deliberate mouse movements
- Keep narration clear and confident
- Consider background music (subtle, upbeat)
